# Tai Say Company eTracking

React web application for Tai Say Company eTracking, built with Vite, Tailwind CSS, React Router, and Lucide React.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Structure

- **Header**: Blue-900 background, white title "Tai Say Company eTracking"
- **Navigation**: Home, eTracking (hover dropdown), Announcement, Report (hover dropdown), Setting
- **Home**: Announcements box with date + clickable title list
- **Announcement**: Add Announcement button and placeholder for announcement details
